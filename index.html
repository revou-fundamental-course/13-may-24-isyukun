Luas Segitiga
Rumus Luas Segitiga yaitu :
L = 1/2 x a x t
Dimana :
L = Luas
a = Panjang Alas
t = Tinggi
Hitung Luas Segitiga

L = 1/2 x a x t
L = 1/2 x 'var a' x 'var t'
L = 'var luas'
Keliling Segitiga
Rumus Keliling Segitiga yaitu :
K = a + b + c
Dimana :
K = Keliling
a = Sisi 1
b = Sisi 2
c = Sisi 3
Hitung Keliling Segitiga


K = a + b + c
K = 'var a' + 'var b' + 'var c'
K = 'var keliling'
